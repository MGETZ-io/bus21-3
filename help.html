Wie funktioniert der Boardcomputer?

1. Anmeldung

Um den Boardcomputer zu verwenden, melden Sie sich mit Ihrer Fahrernummer an. Geben Sie Ihre Nummer ein und tippen Sie auf „Anmelden“.

2. Hauptmenü

Nach der Anmeldung gelangen Sie ins Hauptmenü. Dort können Sie die verschiedenen Funktionen des Systems auswählen:

  • Route auswählen: Wählen Sie eine Übungsroute aus, die Sie fahren möchten.
  • Ticketing: Verkaufen Sie verschiedene Fahrkarten (Kinderfahrkarte, normale Fahrkarte, Umsteigerkarte).
  • Notruf: Aktivieren Sie den Notruf, wenn nötig.
  • Ansage: Machen Sie Durchsagen über den Buslautsprecher.

3. IDCS - Routeninfo

Wenn Sie eine Route auswählen, können Sie die aktuellen Haltestellen und die Routenbeschreibung einsehen. Jede Route hat vordefinierte Haltestellen, die Sie durchlaufen werden. Tippen Sie auf „Nächste Haltestelle“, um fortzufahren.

4. Ticketing

Im Ticketing-Bereich können Sie unterschiedliche Ticketarten verkaufen, wie Kinderfahrkarten, normale Fahrkarten oder Umsteigerkarten.

5. Schnellklick-Buttons

Im Hauptmenü finden Sie Schnellklick-Buttons für Notruf und Ansagen, die Sie schnell und einfach aktivieren können.
